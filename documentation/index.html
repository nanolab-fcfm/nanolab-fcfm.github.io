    🏛 NanoLab Documentation

    Welcome to the NanoLab Documentation Hub. Each of our software and hardware projects has its own full documentation site — pick one below to dive in. We believe in open science: explore the code, build the hardware, and reproduce our measurements.

    💻 Software

    • Laser Setup

      Experimental control for laser, I–V and transfer-curve measurements — a PyMeasure-based GUI with a YAML/Hydra configuration system.

      Open documentation → · GitHub

    🔧 Hardware

    • Clicker

      An ESP8266 device that drives the MHP30 hotplate buttons over serial to automate plate-temperature control.

      Open documentation → · GitHub

    • Temperature Sensor

      A custom serial temperature sensor (PT100 via MAX31865 + DS18B20) for plate and ambient readings, used by Laser Setup.

      Open documentation → · GitHub

    📝 Protocols & methods

    Step-by-step measurement protocols live alongside the software that runs them — see the Lab Protocols in the Laser Setup documentation.

    🤝 Open science & contributing

    At NanoLab we believe in the power of open science. All of our projects are public: browse the code, file issues, or submit improvements on GitHub 🚀 — each project's documentation includes setup and usage instructions.
