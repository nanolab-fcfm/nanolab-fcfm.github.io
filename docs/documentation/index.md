# 🏛 NanoLab Documentation

Welcome to the **NanoLab Documentation Hub**. Each of our software and hardware
projects has its own full documentation site — pick one below to dive in. We
believe in **open science**: explore the code, build the hardware, and reproduce
our measurements.

## 💻 Software

<div class="grid cards" markdown>

-   :material-laser-pointer: **[Laser Setup](https://nanolab.cl/laser_setup/)**

    ---

    Experimental control for **laser, I–V and transfer-curve** measurements — a

    [**Open documentation →**](https://nanolab.cl/laser_setup/) ·
    [GitHub](https://github.com/nanolab-fcfm/laser_setup)

## 🔧 Hardware

<div class="grid cards" markdown>

-   :material-gesture-tap-button: **[Clicker](https://nanolab.cl/clicker_instrument/)**

    ---

    An ESP8266 device that drives the **MHP30 hotplate** buttons over serial to
    automate plate-temperature control.

    [**Open documentation →**](https://nanolab.cl/clicker_instrument/) ·
    [GitHub](https://github.com/nanolab-fcfm/clicker_instrument)

-   :material-thermometer: **[Temperature Sensor](https://nanolab.cl/temperature_sensor/)**

    ---

    A custom **serial temperature sensor** (PT100 via MAX31865 + DS18B20) for
    plate and ambient readings, used by Laser Setup.

    [**Open documentation →**](https://nanolab.cl/temperature_sensor/) ·
    [GitHub](https://github.com/nanolab-fcfm/temperature_sensor)

</div>

## 🤝 Open science & contributing

At **NanoLab** we believe in the power of open science. All of our projects are
public: browse the code, file issues, or submit improvements on
**[GitHub](https://github.com/nanolab-fcfm)** 🚀 — each project's documentation
includes setup and usage instructions.
